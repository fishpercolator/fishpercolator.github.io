Open source

We're huge fans of the open source movement, and we use open source code every single day we are building software.

Open source only works when people give back, and we give back as often as we can. Some of the contributions we're most proud of include:

  • Many contributions to fog.io that resulted in Quinn being named the project's MVP for release 1.37
  • Adding an audit mode to the Spinach testing tool
  • Adding authorization features to the Administrate library
  • Adding support for editing polymorphic fields to Administrate
  • A small but significant contribution to the core of Rails itself
  • A gem that adds latitude and longitude field types to Administrate
  • Fixing a Linux-specific issue with Workrave, a C++ app that helps prevent RSI
